# Vue Web UI Template

Detta är ett GitHub‑template‑repo för admin‑gränssnittet i plattformen.

## Funktioner
 Vue 3 + Vite
 Vuetify 3 (admin UI)
 Pinia stores
 Vue Router
 OpenAPI-DTO-typer (generated.d.ts)
 Docker‑image (Nginx)
 ESLint
 VERSION‑fil

## Utveckling
```bash
npm install
npm run dev
```

## Bygg
```bash
npm run build
```
